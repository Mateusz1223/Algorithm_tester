# Algorithm_tester

Program which tests alghorithms

## Usage
Windows:
`tester (program to be tested) (tests directory) -h to hide fault message (optional)`

Linux:
`./tester (program to be tested) (tests directory) -h to hide fault message (optional)`

Example:

Windows:
`tester example\add.exe example\tests`

Linux:
`./tester example/add example/tests`
